Simple and powerful API for NGL message automation

Endpoint

Base URL: http://localhost:5000
GET /nglspam
Send multiple anonymous messages to an NGL user.
username *required — NGL username amount *required — Number of messages (1-40) message *required — Message to send
GET /nglspam?username=mythiannj&amount=5&message=Hello%20there!
{ "success": true, "message": "Successfully sent 5 messages to mythiannj.", "sent": 5, "failed": 0, "total": 5 }
